How to Use Different Modules in the POS System

Navigation Tip: You can access all modules using the main navigation menu at the top of each page, or by clicking the buttons below.

1. Sales Module

The Sales module is where you process customer transactions and generate invoices.

Key Features:
  • Search and add products to invoices
  • Calculate GST (SGST/CGST) automatically
  • Process payments and generate receipts
  • View transaction history

2. Purchase Module

The Purchase module handles incoming stock from suppliers and manages purchase orders.

Key Features:
  • Create and manage purchase orders
  • Record incoming stock
  • Track supplier information
  • Update inventory levels

3. Inventory Module

The Inventory module provides real-time visibility into stock levels and helps manage inventory.

Key Features:
  • View current stock levels for all products
  • Identify low stock items
  • Search and filter inventory
  • Refresh inventory data

4. Reports Module

The Reports module provides insights into business performance through various reports.

Key Features:
  • Generate sales summary reports
  • View top-selling products
  • Analyze sales trends
  • Export reports to PDF/Excel

5. Admin Module

The Admin module provides system management capabilities (Admin access required).

Key Features:
  • Manage products (add, edit, delete)
  • Manage users and roles
  • Configure system settings
  • View system logs

Note: This module is only accessible to users with Admin privileges.

6. Dashboard

The Dashboard provides an overview of the system and quick access to all modules.
